# AI-prototype-bot

 **Зарегистрируйтесь на AssemblyAI**, чтобы получить бесплатный API-ключ и $50 на кредиты.
 
Этот проект показывает, как сделать голосового ИИ-бота в реальном времени на **Python, AssemblyAI и ElevenLabs**.  
Бот умеет принимать звонки, расшифровывать речь, генерировать интеллектуальные ответы и вести диалог, как настоящий человек.  
Идеально подходит для колл-центров, службы поддержки и виртуальных ресепшенов.

---

##  Возможности

- **Реальная транскрипция в реальном времени**: AssemblyAI преобразует речь в текст максимально точно.  
- **Обработка естественного языка**: OpenAI генерирует ответы с учётом контекста разговора.  
- **Синтез голоса ИИ**: ElevenLabs превращает текстовые ответы в естественный голос.  

---

##  Цель проекта

Этот бот работает как **виртуальный ресепшн**:

- Принимает звонки в реальном времени  
- Понимает речь и контекст  
- Генерирует умные ответы через OpenAI  
- Говорит естественным человеческим голосом через ElevenLabs  
- Хранит историю диалога для плавного общения  

Можно адаптировать под любую сферу: продажи, техподдержка, запись клиентов, холодные звонки и т.д.

## Установка

### 1. Установка зависимостей (пример для MacOS)
```bash
brew install portaudio
brew install mpv

pip install "assemblyai[extras]"
pip install elevenlabs==0.3.0b0
pip install --upgrade openai
```

### 2. Настройка API ключей
```bash
Вставьте свои ключи в код:
aai.settings.api_key = "ASSEMBLYAI_API_KEY"
self.openai_client = OpenAI(api_key="OPENAI_API_KEY")
self.elevenlabs_api_key = "ELEVENLABS_API_KEY"
```

### 3. Как запускать

Убедитесь, что подключен микрофон и установлены PortAudio и mpv.
Запустите скрипт Python:
```
python main.py
```
Бот проиграет приветствие:
```
“Спасибо, что позвонили в .... Меня зовут ..., чем могу помочь?”
```
После этого начнётся транскрипция речи в реальном времени. Говорите в микрофон — бот будет отвечать голосом.

### 4. Настройка роли бота
Роль бота задаётся через system prompt:
```
{"role":"system", "content":"Вы являетесь ресепшионистом стоматологической клиники. Действуйте эффективно и сообразительно."}
```
Меняйте текст для других сценариев:
- Продавец
- Оператор поддержки
- Холодный звонок
- Виртуальный ассистент

### 5. Возможные улучшения
